# Data Warehouse para Gestión de Mermas en Supermercados

Proyecto de Business Intelligence orientado al análisis y control de mermas operativas en supermercados.

## Objetivo

Diseñar una solución de Data Warehouse que permita identificar pérdidas económicas, analizar causas de merma y generar información para la toma de decisiones.

## Tecnologías utilizadas

- MySQL
- Excel
- ETL
- Data Warehouse
- Business Intelligence

## Funcionalidades

- Integración de datos mediante procesos ETL.
- Consolidación de información proveniente de distintas fuentes.
- Modelado dimensional.
- Generación de indicadores de gestión.
- Dashboard para visualización de resultados.
- Análisis de pérdidas por sucursal, categoría y producto.
